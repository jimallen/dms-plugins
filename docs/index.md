## Quick Reference

### MeetingWidget Plugin

- **Type:** Desktop Widget Plugin
- **Purpose:** Google Calendar integration with meeting display
- **Tech Stack:** QML, Qt Quick, Quickshell, Google Calendar API
- **Root:** `MeetingWidget/`

| File | Purpose |
|------|---------|
| `MeetingWidget.qml` | Bar widget with countdown |
| `MeetingsTab.qml` | Dashboard tab with meeting list |
| `MeetingWidgetSettings.qml` | Settings panel |
| `GCalService.qml` | Calendar data service |

### CenterWidget Plugin

- **Type:** Desktop Widget Plugin
- **Purpose:** Time, date, and weather display with dynamic colors
- **Tech Stack:** QML, Qt Quick, Quickshell, WeatherService
- **Root:** `CenterWidget/`

| File | Purpose |
|------|---------|
| `CenterWidget.qml` | Bar widget with time/date/weather |
| `CenterWidgetSettings.qml` | Settings panel |

## Getting Started

### For Users

**MeetingWidget:**
1. Configure Google OAuth credentials
2. Run `dms gcal auth`
3. Enable plugin in Settings -> Plugins
4. Add MeetingWidget to DankBar

**CenterWidget:**
1. Enable plugin in Settings -> Plugins
2. Add CenterWidget to DankBar
3. Configure weather in DMS Settings if desired

### For Developers

1. Clone repository
2. Symlink or configure plugin scan path
3. Edit QML files - no build required
4. Reload Quickshell to test changes

See [Development Guide](./development-guide.md) for detailed instructions.

## CLI Commands (MeetingWidget)

```bash
dms gcal auth           # Authenticate with Google
dms gcal status         # Check connection status
dms gcal logout         # Disconnect account
dms gcal events         # Fetch events (debugging)
```
